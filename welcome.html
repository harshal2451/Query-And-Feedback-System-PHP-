Introduction

To better improve student knowledge this system is introduced. A class of 60 minutes would not be enough to answer every question. “What if a students can ask a question outside the class ?” or “What if a student can get answer even after class ?” or “What if a faculty can give answers for appropriate question outside the class and also could improve one’s skill ?” or “How about HOD(Head Of Department) view staff performance based on student reviews ?”. The Query and Feedback System would provide improvement in student knowledge and ease in studies , this system would have question and answer portal for student queries and faculty could review one’s self and improve his/her teaching skills.
